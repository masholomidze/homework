რა არის html?

html არის სტანდარტული მარკირების ენა დოკუმენტებისთვის, რომლებიც შექმნილია ვებ ბრაუზერში გამოსატანად. ის განსაზღვრავს ვებ კონტენტის შინაარსს და სტრუქტურას.

რა არის css?

css არის სტილის ენა, რომელიც გამოიყენება მარკირების ენაზე დაწერილი დოკუმენტის პრეზენტაციისა და სტილის დასაზუსტებლად

რა არის js

js(java script) არის პროგრამირების ენა და ვებ-გვერდის ძირითადი ტექნოლოგია, HTML და CSS-თან ერთად.
